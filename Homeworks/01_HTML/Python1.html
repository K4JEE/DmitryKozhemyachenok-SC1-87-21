Из множества существующих языков програмирования Python- один из самых легких в изучении. Он создан в конце 80-х годов ХХ века, но с тех пор он сильно изменился. Хотя Python входит в состав большинства дистрибутивов Linux, на него часто не обращают внимания при выборе языка для изучения. В этой статье мы рассмотрим в создание консольных приложений, а в следующей — программирование графических интерфейсов (GUI). Давайте создадим простую программу прямо сейчас.

Наша первая программа

Наберем в текстово редакторе (например в gedit). Потом мы разберемся что делает каждая строка и пойдем дальше.

Введем эти 4 строки

!/usr/bin/env python
print 'Hello, i am a Python prorgam.'
name= input('What is your name?')
print 'Hello there,' + name + '!'

Это всё. Сохраните файл как hello.py в любом месте. Я предлагаю сохранить его в папке python_examples в домашнем каталоге. Этот пример показывает, как просто написать программу на Python. Для запуска программы нужно сделать её файл исполняемым. Для этого введите в терминале

chmod +x. hello.py

в папке, в котрой вы сохранили программу. Теперь запустите её.

grag@earth:∼/python_examples$ ./hellp.py
Hello. I am a python program.
What is your name? Ferd Burphel
Hello ther, Ferd Burphel
greg@earth:∼;/python_examples$

Вот и всё.

Пример цикла FOR

Теперь давайте рассмотрим пример программирования цикла. Запустите текстовый редактор и наберите следующую программу:

#!/usr/bin/env python
for cntr in range(0,10):
    print cnrt

Не забудьте поставить символ табуляции перед строкой «print cntr». Это важно. в Python для указания на блоки кода используются не круглые «(» и фигурные скобки «{» скобки как в других языках программирования, а отступы.
